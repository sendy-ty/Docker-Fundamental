# Docker Course Documentation

## Deskripsi Repository
Repository ini berisi dokumentasi lengkap hasil pembelajaran Docker yang mencakup konsep dasar, praktik penggunaan Docker, hingga challenge lanjutan. Seluruh dokumentasi disusun berdasarkan lab dan challenge pada Docker Course (Adinusa).

Dokumentasi ditujukan sebagai:
- Referensi praktikum Docker
- Catatan pembelajaran terstruktur
- Portofolio teknis Docker dan DevOps dasar

---

## Tujuan Pembelajaran
Setelah menyelesaikan seluruh lab dan challenge, pengguna diharapkan mampu:
- Memahami konsep container dan Docker Engine
- Mengelola image dan container Docker
- Membuat dan menggunakan Dockerfile
- Mengelola volume, network, dan storage Docker
- Menerapkan logging dan health check
- Menggunakan Docker Swarm dan Portainer
- Melakukan deployment aplikasi menggunakan Docker

## Challenge Lab

### Challenge 1 – HAProxy with Docker
- Menjalankan beberapa web server container
- Konfigurasi HAProxy sebagai reverse proxy
- Penggunaan Docker Network dan Volume
- Akses service berbasis domain host

### Challenge 2 – Deploy Laravel with Docker
- Build image PHP 7.2 dengan Apache
- Otomatisasi composer install, migrate, dan seeding
- Penggunaan entrypoint script
- Deployment Laravel tanpa langkah manual
- Verifikasi akses dashboard dan autentikasi user
